# OS_Simulation_d-algorithme_d-ordonnancement
Simulateur d’ordonnancement de processus en C. Implémente FIFO, SJF, SJRF et RR. Entrée : fichiers de processus avec cycles CPU/E/S. Sortie : textuelle (CSV) ou graphique, avec temps d’attente, restitution, réponse et taux d’occupation CPU. Extensible à d’autres algorithmes.
